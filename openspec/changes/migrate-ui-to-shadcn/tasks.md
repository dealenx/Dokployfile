# Tasks: Migrate to Shadcn Vue

- [ ] **Infrastructure Setup**
  - [ ] Remove `@nuxt/ui` from `package.json` and `nuxt.config.ts`.
  - [ ] Install `shadcn-vue` and its peer dependencies (`radix-vue`, `lucide-vue-next`, `clsx`, `tailwind-merge`).
  - [ ] Configure `tailwind.config.js` (or Tailwind 4 equivalent in CSS) and `tsconfig.json`.
  - [ ] Run `npx shadcn-vue@latest init`.
- [ ] **Core Components Implementation**
  - [ ] Add `Button` via `npx shadcn-vue@latest add button`.
  - [ ] Add `Input` via `npx shadcn-vue@latest add input`.
  - [ ] Add `Card` via `npx shadcn-vue@latest add card`.
  - [ ] Add `Label` and `Form` components.
  - [ ] Add `Dialog` (Modal equivalent).
  - [ ] Add `Badge` and `Alert`.
- [ ] **Page Migration**
  - [ ] Migrate `app/pages/index.vue` to use Shadcn components.
  - [ ] Migrate `app/pages/[owner]/[repo].vue` to use Shadcn components.
  - [ ] Migrate `app/pages/generate.vue` to use Shadcn components.
  - [ ] Update `app/app.vue` or layout if necessary.
- [ ] **Validation and Cleanup**
  - [ ] Verify all UI interactions still work (GitHub search, generation, copy to clipboard).
  - [ ] Remove any leftover Nuxt UI styles or imports.
  - [ ] Run `pnpm test` to ensure no regressions in logic.
